# Block-Diagram-Simulation-of-Induction_Machine

## Requirements
- MATLAB R2017a or later
- Simulink Toolbox

## How to Run
1. Open `Induction_Machine.slx` in MATLAB.
2. Click "Run" to start the simulation.
3. Analyze the results in Scope blocks.
